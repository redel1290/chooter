# ShooterGame v1.0.0

Top-down 2D мультиплеєрний шутер для Android. Локальний Wi-Fi 1v1.

## Як грати

1. Обидва гравці підключені до **однієї Wi-Fi мережі**
2. Один запускає як **Хост** — бачить свій IP
3. Інший обирає **Підключитись** — вводить IP хоста
4. Кожен вибирає зброю, хост вибирає карту → **Почати**
5. Перший хто набрав **5 вбивств** — переміг

## Керування

| Джойстик | Дія |
|---|---|
| Лівий | Рух персонажа |
| Правий (тримати) | Прицілювання |
| Правий (відпустити) | Постріл |

## Зброя

| Зброя | Дамаг | Швидкострільність | Особливість |
|---|---|---|---|
| Пістолет | 20 | Висока | Точний, далекобійний |
| Дробовик | 15×6 | Низька | Розліт, смертельний впритул |

## Карти

- **Арена** — відкритий простір з укриттями-ящиками
- **Лабіринт** — коридори та глухі кути

## Збірка

Пуш в `main` → GitHub Actions автоматично збере APK.
Завантажити: `Actions → Build APK → Artifacts → ShooterGame-v1.0.0-debug`

## Технології

- **LibGDX 1.12.1** — ігровий фреймворк
- **Kotlin** — мова розробки  
- **UDP сокети** — мережева синхронізація (20 пакетів/сек)
- **GitHub Actions** — автоматична збірка APK
